# firememo

A simple memo web application that uses the Firebase Cloud Firestore database.

## Demo
N/A

## Features
- Create, view, and delete memos
- Memos are stored in the Cloud Firestore database

## Requirements
- Firebase account
- Cloud Firestore enabled

## Usage
1. [Register for Firebase](https://firebase.google.com)
2. [Enable Cloud Firestore](https://firebase.google.com/docs/firestore?hl=en)
3. Create a `firebaseConfig.js` file based on the `firebaseConfig_template.js` template
4. Run a web server (e.g., `live-server`) and open `index.html`

## Data / API
This project uses the Firebase Cloud Firestore API to store and retrieve memos.
